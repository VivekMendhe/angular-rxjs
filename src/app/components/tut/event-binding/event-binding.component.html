Click Event

(click): Triggered when a user clicks on an element.

Keydown Event

(keydown): Triggered when a user presses a key on the keyboard.

Mouseup Event

(mouseup): Triggered when the user releases the mouse button over an element.

Mouseover Event

(mouseover): Triggered when the mouse pointer enters an element.

Mouseout Event

(mouseout): Triggered when the mouse pointer leaves an element.

Submit Event

(submit): Triggered when a form is submitted.

Focus Event

(focus): Triggered when an element gains focus.

Blur Event

(blur): Triggered when an element loses focus.

Change Event

(change): Triggered when the value of an element changes.

Scroll Event

(scroll): Triggered when the user scrolls the page.

Load Event

(load): Triggered when the page finishes loading.

Unload Event

(unload): Triggered when a page is unloaded.

Resize Event

(resize): Triggered when the window is resized.

Double Click Event

(dblclick): Triggered when a user double-clicks on an element.

Context Menu Event

(contextmenu): Triggered when the user right-clicks on an element.

Drag Event

(drag): Triggered when an element is being dragged.

Drop Event

(drop): Triggered when a dragged element is dropped onto a target element.

Mouse Enter Event

(mouseenter): Triggered when the mouse pointer enters an element.

Mouse Leave Event

(mouseleave): Triggered when the mouse pointer leaves an element.

Touchstart Event

(touchstart): Triggered when the user touches their screen.

Touchmove Event

(touchmove): Triggered when the user moves their finger on the screen.

Keydown Specific Key Event

(keydown.enter): Triggered when a specific key is pressed (e.g., Enter).

Keyup Specific Key Event

(keyup.enter): Triggered when a specific key is released (e.g., Enter).

Input Event

(input): Triggered when there’s an input into a form field.

Window Scroll Event

(window:scroll): Triggered when the user scrolls the window.

Window Resize Event

(window:resize): Triggered when the window is resized.

Keypress Event

(keypress): Triggered when the user presses a key on the keyboard (deprecated in favor of keydown and keyup).

Keyup Event

(keyup): Triggered when the user releases a key on the keyboard.

Mousedown Event

(mousedown): Triggered when the mouse button is pressed over an element.

Mousemove Event

(mousemove): Triggered when the mouse is moved within an element.
